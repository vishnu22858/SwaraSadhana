SwaraSadhana - Final package (styled + auth + protected routes)

Run:
1. Start MongoDB
2. Backend:
   cd backend
   python -m venv venv
   # Windows:
   venv\Scripts\activate
   pip install -r requirements.txt
   python seed.py
   python app.py
3. Frontend:
   cd frontend
   npm install
   npm run dev
   Open http://localhost:5173
